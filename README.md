## 🛠 What is PrivHound?

PrivHound helps you understand how local attacks on Windows systems can happen. It builds a map of possible attack paths that attackers might use to gain higher access rights on your computer. The tool uses data from Windows systems and shows how different pieces connect to create risks.

This application is useful if you want to:

- See how local privilege escalation can occur in Windows.
- Get a clear picture of your system’s security from an attacker's point of view.
- Find weak points and plan ways to protect your computer.

PrivHound works by collecting information and turning it into an easy-to-understand graph. This graph shows how different accounts and permissions link together, helping you spot where risks exist.

## 🧭 How to Use PrivHound

After running the program, the main window will open. You will see options to start collecting Windows data.

1. Click **Start Scan** to begin collecting your system’s local security information.
2. The scan may take a few minutes depending on your system.
3. When done, PrivHound will create a visual graph. This graph shows accounts, groups, and how permissions link.
4. Use the navigation tools to zoom in and out or move around the graph.
5. Look for paths where a low-level user can jump to an admin account. These show potential risks.
